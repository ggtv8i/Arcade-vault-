## 🚀 رفع الموقع على GitHub Pages

### الخطوة 1 — إنشاء المستودع
```
1. اذهب إلى github.com وسجّل دخولك
2. اضغط "New repository"
3. اسمه: arcade-vault  (أو أي اسم تريد)
4. اجعله Public
5. اضغط "Create repository"
```

### الخطوة 2 — رفع الملفات
```
اسحب الملفات التالية وأفلتها في مستودعك:
  ├── index.html        ← الموقع الرئيسي
  ├── README.md         ← هذا الملف
  └── roms/             ← مجلد ألعابك (اختياري)
      ├── game1.nes
      ├── game2.gba
      └── ...
```

### الخطوة 3 — تفعيل GitHub Pages
```
1. اذهب إلى Settings في مستودعك
2. اضغط "Pages" من القائمة الجانبية
3. في Source اختر: Deploy from a branch
4. اختر branch: main
5. اختر folder: / (root)
6. اضغط Save
```

### الخطوة 4 — موقعك جاهز! 🎉
```
رابط موقعك سيكون:
https://[اسم-المستخدم].github.io/arcade-vault/
```

## ☁️ إعداد مستودع الألعاب السحابي

بعد رفع الموقع، اذهب إلى **الإعدادات ← GitHub Cloud**:

| الحقل | المثال |
|-------|--------|
| المالك | `yourusername` |
| المستودع | `arcade-vault` |
| الفرع | `main` |
| مجلد الروم | `roms` |
| Token | (اختياري — للمستودعات الخاصة) |

ثم اضغط **"حفظ الإعدادات"** → **"جلب الألعاب الآن"**

الألعاب ستظهر تلقائياً في المكتبة لكل زائر! 🎮
